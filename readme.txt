Небольшой гайдик, как все это дело запустить

config для сервера лежит тут js/server/config.js
config для клиента лежит тут js/client/config.js

1) Что бы стартануть всю эту катавасию надо во-первых загрузить nodejs
    для линукса - dnf/yum/apt-get install nodejs
    для линукса - dnf/yum/apt-get install npm
    для винды и всего остального - https://nodejs.org/en/download/
        для винды вроде бы как сразу устанавливается...

2) дальше выполнить
   cd js/server ( Это значит зайти в папку сервера )
   node main.js ( Это стартует сервер )

3) Разумеется нихрена не заработает, потому что не загружены модули.
Так вот. Устаналиваем модули:
    npm install websocket
    npm install request

4) После этих манипуляций север обязан стартануть.
5) Теперь самое сложное, нужно зарегать на себя приложуху на сайте разработчиков
    Идем вот сюда: https://developers.eveonline.com/
    Логинимся, если не залогинены.
    Дальше идем на вкладку Applications
    Жмем CREATE NEW APPLICATION
    Заполняем все поля, как надо (имя и описание вообще насрать как)
    Дальше идет CONNECTION TYPE, там ставим галку на Authentication & API Access
    Дальше идет выбор доступа на разные категории запросов, мы должны выбрать вот эти:
       - esi-location.read_location.v1
       - esi-location.read_ship_type.v1
       - esi-clones.read_clones.v1
       - esi-universe.read_structures.v1
       - esi-location.read_online.v1

    В коллбэк url мы пишем, наш адрес, на который будет переадресовано после удачной авторизации
    Окей. Мы создали приложение, теперь открываем js/server/config.js и заполняем все как положено.


6) Включите VPN, потому что сраный РКН все переблокировал к херам.
